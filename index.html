Usage

This is a tool for working out multi-component solution recipes.

Add a row for each component of the desired solution, indicating the stock and desired concentrations of each. The volume of each component will be computed to produce the desired concentrations in a solution of the target volume.

You can use the ↑ and ↓ arrow keys to move between rows. Tab and Shift + Tab will move between fields. Alt + A will add a row. Alt + D will remove a row.
